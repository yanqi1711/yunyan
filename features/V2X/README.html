测试

  1. 查看模组状态，是否为active，若模组正常返回active，则表示激活成功
# 进入v2x系统, 密码root
ssh root@192.168.62.199
# 使用athelper工具
athelper
# 查询v2x状态
at+irms

  1. 查看v2x波特率，出厂波特率设置为115200，若不是115200，则需要修改
# 进入v2x系统, 密码root
ssh root@192.168.62.199
# 查看v2x波特率
stty -F /dev/ttyS1

  1. 查看模组是否正常发送BSM包
# 进入v2x系统, 密码root
ssh root@192.168.62.199
# 查询是否在发BSM包
v2xdump -Q out

修改波特率方法

方法一：使用命令修改

# 进入v2x系统, 密码root
ssh root@192.168.62.199
# 修改波特率为115200
stty -F /dev/ttyS1 115200
